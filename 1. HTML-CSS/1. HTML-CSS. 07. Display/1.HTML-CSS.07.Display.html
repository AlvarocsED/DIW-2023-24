Explicación de los valores de la propiedad CSS inline:

  • inline: Los elementos con display: inline se comportan como parte del flujo de texto y sólo ocupan el espacio necesario para su contenido. No puedes establecer ancho o alto en elementos inline. Tampoco le afectan los márgenes en vertical, pero sí en horizontal.
  • inline-block: Los elementos con display: inline-block se comportan como elementos inline, pero permiten establecer un ancho y alto específicos. Pueden aparecer en la misma línea y ocupan espacio según su contenido y dimensiones definidas.
  • block: Los elementos con display: block son bloques independientes que ocupan todo el ancho disponible en su contenedor. Cada elemento block comienza en una nueva línea y puedes definir su ancho y alto.

Otro aspecto importante de los elementos inline es que los posibles valores de padding o border no afectan a las dimensiones de su padre. Es decir, si aumentas el padding o el border de un elemento inline, su padre no se adaptará al cambio de dimensiones.

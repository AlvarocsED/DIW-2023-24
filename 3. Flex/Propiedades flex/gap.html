  • La propiedad row-gap establece la separación entre filas si flex-direction: column o si estamos en un contenedor multilínea (flex-wrap: wrap).

  • La propiedad column-gap establece la separación entre columnas si flex-direction: row o si estamos en un contenedor multilínea (flex-wrap: wrap).

  • La propiedad gap permite estableces row-gap y column-gap al mismo tiempo. Si se le pasa un valor (ej: gap: 10px) establece el mismo valor a ambas propiedades. Si se le pasan dos valores el orden utilizado será gap: <row-gap> <column-gap>.

Las tres propiedades admiten valores dados en porcentajes. Tal y como indica la especificación de W3C, cuando se indica un valor en porcentaje este se resuelve computando el tamaño del content box del contenedor. Esto lo puedes comprobar en este mismo ejemplo utilizando las DevTools del navegador. Como hemos indicado en el CSS que el contenedor sea de 160px de ancho y con un padding de 20px, la anchura del content box es de 160-20-20 = 120px. Si pones column-gap: 10% el navegador calculará el 10% de 120px, 12px, y lo aplicará.
